# 🌱 GreenRoot

A sustainable living app that helps you track your carbon footprint 
and build better habits. Built with React and TypeScript on the frontend, 
with a Python FastAPI backend.

## Built with
- React 18
- TypeScript
- Python FastAPI
- Vite

## Pages
- Home
- Features
- Pricing

## Live Demo

## Run frontend locally
cd frontend
npm install
npm run dev

## Run backend locally
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
